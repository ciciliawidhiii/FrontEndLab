## Setting Up Project

### Install required dependencies

  - *Frontend*
    - Ubah directory menjadi frontendlab 
      - buka terminal
      - `cd frontendlab`
    - `npm install`

### Run the program

  - *Frontend*
    - Ubah directory menjadi frontendlab
      - buka terminal
      - `cd frontendlab`
    - `npm start`
